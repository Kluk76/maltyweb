Quand un brassin ouvre pour la première fois sur (recette, lot), le système déclenche automatiquement une cascade de création en trois temps — sans intervention opérateur.

La mother shell EMB 244 apparaît immédiatement dans la zone Brasserie du tableau — heartbeat vert, session active, cuve affectée.

Cette cascade s'exécute aussi quand un fermenting / racking / packaging child s'ouvre pour un (recipe, batch) qui n'a pas encore de mother — le système retro-lie automatiquement. La logique vit dans session_open() · Point d'entrée unique · Transaction atomique · Pas d'intervention opérateur · Idempotent si la mother existe déjà (pas de doublon créé).
